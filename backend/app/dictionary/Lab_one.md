# Lab Ketiga – Praktikum Hukum Coulomb

## Komponen Lab

| Komponen | Keterangan |
|---|---|
| **Penembak Muatan** | Tarik ke bola konduktor, lepas saat ujung masuk bola untuk ubah muatan |
| **Tombol Muatan +/-** | Merah = positif, Biru = negatif. Klik sebelum menembak |
| **Slider Muatan** | Tidak aktif di Kegiatan 1 |
| **Bola Konduktor 1** | Kiri, statis. Merah = +, Biru = − |
| **Bola Konduktor 2** | Kanan, bisa bergerak kiri/kanan sesuai interaksi muatan. Merah = +, Biru = − |
| **Tiang Statif** | Penggantung Bola 2, tidak bisa digeser |
| **Tembok Penggaris + Laser** | Di atas. Laser bergerak: kiri hingga −1000 N jika muatan berbeda jenis, kanan hingga +1000 N jika muatan sejenis |
| **Timi Bot** | Chatbot bantuan praktikum |
| **Tombol Reset** | Atur ulang lab (kanan bawah, kuning) |
| **Tombol Lanjutkan** | Lanjut ke sesi refleksi |
