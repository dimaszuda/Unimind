# Lab Ketiga – Praktikum Hukum Coulomb

## Komponen Lab – Kegiatan 2

| Komponen | Keterangan |
|---|---|
| **Penembak Muatan** | Tarik ke bola konduktor, lepas saat ujung masuk bola untuk ubah muatan |
| **Slider Muatan** | Geser kiri = negatif, kanan = positif. Range −10 µC hingga +10 µC, kelipatan 1 µC |
| **Nilai Penembak Muatan** | Angka di kanan penembak, berubah otomatis saat slider digeser |
| **Bola Konduktor 1** | Kiri, statis (tidak bergerak baik manual/otomatis). Merah = +, Biru = − |
| **Bola Konduktor 2** | Kanan, bisa bergerak kiri/kanan otomatis sesuai interaksi muatan tapi tidak dapat digerakan manual atau didekatkan dengan bola konduktor 1. Merah = +, Biru = − |
| **Tiang Statif** | Penggantung Bola 2, tidak bisa digeser |
| **Tembok Penggaris + Laser** | Di atas. Laser bergerak: kiri hingga −1000 N jika muatan berbeda jenis, kanan hingga +1000 N jika muatan sejenis |
| **Timi Bot** | Chatbot bantuan praktikum |
| **Tombol Reset** | Atur ulang lab (kanan bawah, kuning) |
| **Tombol Lanjutkan** | Lanjut ke sesi refleksi |
